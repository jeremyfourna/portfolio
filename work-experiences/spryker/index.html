Features & Projects I worked on

Personalisation Engine
To personalise an experience on one touch-point is already a difficult task, so imagine building an engine to allow a brand to centralise the personalisation of their experience across any number of touch-points (web, apps, email, PoS, Social Networks, etc.) and for millions of users.
I am currently writing a concept about how such a personalisation engine would work with the Spryker OS.
Split Delivery
Allow a customer to be delivery to multiple addresses and/or to multiple delivery dates.
Return Management
The return process should be as easy as possible and should provide to the customer as much option as possible to return/exchange the product.
